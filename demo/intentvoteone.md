# Intent: Vote Initiation App

## Objective

Create a new directory named:

    vote-initiation-app

Generate a complete React + java application using Vite. This application shal be used by International Monitory Fund (IMF) for the members to vote on resolution with following metadata
 - Voting deadline: March 25th 2026
 -  Title of Resolution: "Fiscal Responsibility-2026" 
 -  Description of Resolution: "The IMF is advising countries to implement growth-friendly fiscal adjustments to reduce high public debt, which is expected to rise further"
 - Eligible Member Countries: USA, Canada, UK, Netherlands, UAE, Australia

 

---

## Functional Requirements 
### 1. UI

- Display 6 entries under one radio button group that corresponds to Eligible Member Countries
- User should be able to login to the application with username "admin" and password "admin"
- User can only select one of the 6 entries in radio button group
- Have submit and cancel buttons at the bottom
- Cancel button shall reset the page to initial state
- Submit button will send the user selected option to the Backend
---
